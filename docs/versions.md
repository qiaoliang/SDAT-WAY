## 版本策略

### 发布周期

- **主要版本**：每6-12个月
- **次要版本**：每2-4周
- **补丁版本**：按需发布

### 支持策略

- **当前版本**：完全支持
- **上一个主要版本**：维护支持
- **更早版本**：不再支持

### 升级建议

- 建议使用最新版本
- 主要版本升级需要测试
- 补丁版本可以安全升级
